# Movie Vault 🎬

## About
Movie Vault is a modern OTT-style Flutter web application that allows users to browse and stream movies.  
It provides a seamless and responsive interface with support for:
- Movie listing with details (title, rating, description)
- YouTube trailer/video playback (works on Web & Mobile)
- Episode-based streaming for series
- Continue Watching & Trending Movies
- Firebase Firestore backend for storing movie data

The app is built using **Flutter**, **Firebase**, and **YouTube IFrame Player**, and is fully deployable on the web.
